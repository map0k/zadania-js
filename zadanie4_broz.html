1)Kiedy powstał JavaScript (podaj datę)

Język JavaScript powstał 4 grudnia 1995 roku.

2)Dlaczego jest to język skryptowy

Ponieważ nie musi zostać skompilowany, aby można było zobaczyć efekty jego działania.

3)Gdzie jest wykonywany JS (klient lub serwer)

JavaScript jest wykonywany na przeglądarce klienta.

4)Dlaczego JS to bezpieczne narzędzie

Jest on natywnie obsługiwany przez przeglądarki internetowe przez co można go wykorzystywać bez konieczności instalowania dodatkowych narzędzi.

5)Czy jest to język obiektowy, jeśli tak to dlaczego

Tak. JS jest językiem obiektowym, ponieważ wykorzystuje obiekty zawierające dane i funkcje.

6)Wymień zalety JS:

Zaleta1 - Szerokie możliwości zastosowania.
Zaleta2 - Wysoka kompatybilność z przeglądarkami ze względu na popularność.
Zaleta3 - Prostota użycia, i pisania w nim programów.

7)Framework to:

Jest to program do tworzenia innych programów w JavaScript. Zawiera predefiniowane kody i biblioteki ułatwiające tworzenie i zarządzanie nimi.

8)Do aplikacji mobilnych można użyć Framework

Tak. Do aplikacji mobilnych również można użyć Framework.

9)Aplikacje responsywne to:

Aplikacje, które dostosowują się do różnych urządzeń i rozdzielczości ekranu co pozwala na używanie aplikacji niezależnie od tego z jakiego urządzenia się z nich korzysta.

10)Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu

<input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">
Podany fragment kodu tworzy przycisk, który otwiera nową stronę.

11)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu

window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
Podany fragment kodu otwiera okno z wyświetlonym plikiem obraz.html.
 toolbar=no -> ukrywa pasek narzędzi
 directories=no -> ukrywa przyciski katalogów
 menubar=no -> ukrywa pasek menu przeglądarki
 height=280 -> ustawia wysokość na 280px
 width=160 -> ustawia szerokość na 160px
 top=200 -> ustawia okno 200px względem góry ekranu
 left=200 -> ustawia okno 200px względem lewej strony ekranu

12)Opisz następującą instrukcję window.close()

Funkcja ta zamyka okno.
